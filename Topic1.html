HTTP History & Early Versions

HTTP (Hypertext Transfer Protocol) is the language that web browsers and servers use to communicate. It allows your browser to request information and servers to respond with web pages, images, videos, and more (Wilde, 2025).

HTTP was created in the late 1980s and early 1990s by Tim Berners-Lee at CERN as part of his invention of the World Wide Web. He needed a way for computers to share documents using hyperlinks, which led to the creation of the first HTTP version, HTTP/0.9 in 1991. This version was extremely simple, only handling basic text and allowing one-line requests.

As more people used the internet, the need for more complex functionality arose. HTTP/1.0, introduced in 1996, supported different file types like images and videos, headers for extra information, and error codes such as 404 Not Found. These changes allowed websites to become interactive and support multimedia content (Wikipedia, 2025).
