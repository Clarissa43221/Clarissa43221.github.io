Key Concepts

  • HTTP: Protocol for browser-server communication.
  • HTTPS: Secure HTTP using encryption (SSL/TLS).
  • Persistent Connections: Reuse connections to speed up websites.
  • Multiplexing: Multiple requests over one connection (HTTP/2).
  • QUIC: Transport protocol for HTTP/3 to reduce latency.
  • Headers: Extra info sent with requests/responses.
  • Error Codes: Messages like 404 Not Found.
