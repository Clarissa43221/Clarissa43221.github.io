Description

The internet is built on layered technologies that enable computers to communicate. One of the most important is the Hypertext Transfer Protocol (HTTP). Whether streaming video, clicking links, or shopping online, HTTP handles the requests and responses that make the web possible. This site summarizes the origins, evolution, and current/future directions of HTTP.

Key Terms

  • HTTP — Hypertext Transfer Protocol
  • HTTPS — HTTP over TLS/SSL (secure)
  • TCP/IP — Core transport/networking stack
  • QUIC — UDP-based transport protocol used by HTTP/3
  • Multiplexing — Sending multiple requests simultaneously
